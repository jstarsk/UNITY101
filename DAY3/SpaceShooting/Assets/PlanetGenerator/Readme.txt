Use included textures as inputs for material, or create ones with PlanetMapsGenerator substance. 
You can drag and drop generated textures in corresponding slots in material. 
If needed, generated maps can be exported by pressing cog icon in the top right in inspector of procedural material
All substances have specific parameters like size of continents or holes in clouds, so be sure to use it.

To add volumetric atmosphere just duplicate planet gameobject and apply material with Human Unit/Atmosphere shader on the clone.

Use standart unity Post effect stack glow to add atmosphere glow.
